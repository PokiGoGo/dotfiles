# dotfiles

## Setup
``` shell
$ cd ~
$ git clone https://github.com/chronolai/dotfiles.git .dotfiles
```

## .gitconfig
``` shell
$ cp .dotfiles/.gitconfig ~/.gitconfig
$ git config --global user.name "your_name"
$ git config --global useremail "yout_email"
```
